## Contributing

This repository primarily exists as the source for a personal academic website rather than as a public template or library.

Small fixes are welcome, but this is not set up as a general-purpose open-source project. If you want to suggest a correction, the best path is:

1. Open an issue with the problem and a short proposed fix.
2. If needed, open a focused pull request with minimal scope.

Please avoid broad refactors or template-generalization changes unless they are discussed first.
